myClassroom

A Classroom Management System (No-Server)

As a teacher, one of the most frustrating things is the amount of web apps we use for various things. My concept is to have an app that covers everything from lesson plans to parent and student communication. This app was the beginnings of that idea. Although this app is only built with AngularJS, and does not have a back-end, you can see the general idea of the project.

  • Date: June 2017
  • Category: Front-End Development
  • Role: Front-End Developer via AngularJS

Quizlet-Clone

An app to work just as Quizlet.com

This application was designed to be a tool modeled after Quizlet.com. It has the same functionality as Quizlet.com. This app was built with ReactJS on the front-end, and NodeJS, via express, with a MongoDB database. I served as front-end developer on this group project.

  • Date: July 2017
  • Category: Full-Stack Development (AngularJS and NodeJS with PostgreSQL Database)
  • Role: Front-End Developer via ReactJS
